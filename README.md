# DevKeeper — Управление проектами и задачами

**DevKeeper** — это гибкое десктопное приложение для организации любых проектов и задач. Создавайте проекты, разбивайте их на задачи и подзадачи, отслеживайте статусы, используйте шаблоны и быстро находите нужное.

> 📌 Это **сетевая версия** для совместного использования. Сейчас находится в стадии разработки.

---

## 🚀 Возможности

- **Проекты** – создавайте проекты для любых целей (работа, учёба, дом, хобби). Для проектов с кодом можно указать путь к папке – тогда при добавлении автоматически распарсятся технологии из файла `requirements.txt` (они появятся в виде тегов и будут участвовать в поиске).
- **Задачи** – неограниченная вложенность, описание, заметки, сниппеты кода (если нужно).
- **Статусы** – `Новое`, `В процессе`, `Готово`, `Архивное`. Статус родительской задачи обновляется автоматически в зависимости от статусов подзадач.
- **Избранное** – закрепляйте важные проекты и задачи вверху списка.
- **Шаблоны** – создавайте готовые шаблоны задач и добавляйте их в проект одним кликом.
- **Поиск** – быстрый поиск по названию проекта и технологиям.
- **Горячие клавиши** – ускоряйте работу: `Ctrl+P` (новый проект), `Ctrl+T` (новая задача), `Ctrl+=` (развернуть все задачи), `Ctrl+-` (свернуть все задачи), `Delete` (удаление).
- **Темы** – светлая, тёмная, системная.
- **Интеграция с PyCharm** – при указании пути к исполняемому файлу можно открывать проект прямо из приложения (полезно для разработчиков).
- **Совместная работа над проектами и задачами**

---

## 🔜 Планируемые возможности (в разработке)

- **Совместная работа** – несколько пользователей смогут видеть и редактировать одни и те же проекты.
- **Централизованное хранение** – данные будут храниться на общем сервере.
- **Авторизация** – пользователи будут входить под своими учётными записями.
- **Реальное время** – изменения будут синхронизироваться между участниками.
- **Управление доступом** – возможность делиться проектами с другими пользователями.

## 📦 Установка

На данный момент установщик не готов. Вы можете запустить сетевую версию из исходного кода.

---
## 🛠️ Для разработчиков

- Для сборки .exe используйте эту команду:
    ```bash
    pyinstaller --onedir --windowed --icon=main_icon.ico --name "DevKeeper" --add-data "data;data" --add-data "main_icon.ico;." main.py
    ```
- Но также можно просто запустить файл .spec:
    ```bash
    pyinstaller DevKeeper.spec
    ```

## 📂 Структура данных
- В текущей разработке – используется локальная SQLite (%APPDATA%\DevKeeper\projects.db) для удобства тестирования.
- В будущем – данные будут храниться на центральном сервере (планируется PostgreSQL). Шаблоны задач также будут синхронизироваться.
- Логи – %APPDATA%\DevKeeper\logs\YYYY-MM-DD.log.
